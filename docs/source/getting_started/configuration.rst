Configuration
=============

The configuration presents as a `json <https://www.ecma-international.org/publications-and-standards/standards/ecma-404/>`_ file and divides into 7 major components.

.. code-block:: none

    # Configuration structure
    {
        'algorithm': ...
        'instance': ...
        'executor': ...
        'function': ...
        'method': ...
        'output': ...
        'backdoors': ...
    }
