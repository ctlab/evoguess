Backdoors
=========

Initial list of backdoors.

.. code-block:: none

    'backdoors': [<backdoor>, ...]

Base backdoor
-------------

If the **_list** parameter is not specified, then the **input_set** of the current instance will be used as a set of variables.

.. code-block:: none

    {
        'slug': 'backdoor:base',
        '_list': <optional list of numbers>
    }
